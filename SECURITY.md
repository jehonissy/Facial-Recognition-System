# Security Policy

## Supported Versions

  Version   Supported
  --------- -----------
  1.x       ✅ Yes
  \<1.0     ❌ No

## Reporting a Vulnerability

If you discover a security vulnerability, please do not disclose it
publicly.

Instead:

1.  Describe the issue clearly.
2.  Include reproduction steps.
3.  Mention the affected module or file.
4.  Provide logs or screenshots if available.

The issue will be reviewed and fixed as soon as possible.

Thank you for helping improve the security of this project.
