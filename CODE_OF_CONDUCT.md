# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive
environment for everyone.

## Expected Behavior

-   Be respectful and constructive.
-   Welcome feedback and different viewpoints.
-   Help others learn and collaborate.
-   Report issues responsibly.

## Unacceptable Behavior

-   Harassment or discrimination.
-   Personal attacks or abusive language.
-   Spam or disruptive behavior.

## Enforcement

Project maintainers may remove comments, issues, or contributions that
violate this Code of Conduct.

Thank you for helping create a positive community.
