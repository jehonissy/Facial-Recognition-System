# Changelog

All notable changes to this project are documented here.

## \[1.0.0\] - 2026-07-14

### Added

-   Real-time facial recognition using OpenCV and face_recognition
-   Multi-image enrollment from webcam or image folder
-   Attendance logging to CSV
-   Unknown face snapshot capture
-   Tkinter GUI
-   Command-line interface
-   Unit tests with pytest
-   GitHub Actions CI workflow
-   Docker support
-   Comprehensive documentation

### Changed

-   Improved project structure and code readability.
-   Added logging and configuration management.

### Fixed

-   Improved error handling for webcam, face detection, and missing
    files.
